# Electives

**Required:** ~875 credit hours (~7 Nanodegrees) from 13+ curated options.

Electives let you specialize beyond the core PM/AI/business foundation. Plan before enrolling — see [candidates.md](./candidates.md).

## How electives work

1. Complete core modules first (or follow your Udacity learning plan unlock order)
2. Choose 7 Nanodegrees from the approved elective pool
3. Scaffold each elective using the same structure as `core/` (see below)
4. Update [../progress/TRACKER.md](../progress/TRACKER.md)

## Folder template

When you select an elective, create:

```
electives/<nanodegree-slug>/
├── README.md       # Program overview (mirror core program READMEs)
├── SYLLABUS.md     # Build manually or extend init_program_files.py
├── notes/          # Blank sheets — one per course
└── projects/       # Reflection stubs per project
```

Use the same study workflow as core: [GUIDE.md](../GUIDE.md) · [meta/](../meta/) · [blank-sheet template](../notes/templates/blank-sheet.md).

When Udacity adds electives to the init script, run `python3 scripts/init_program_files.py` to auto-scaffold.
